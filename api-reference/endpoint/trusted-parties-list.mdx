---
title: "List Trusted Parties"
api: "GET https://api.magebank.ai/trusted-parties/{type}"
description: "Retrieve all trusted parties of a specific type"
---

## Request Headers

<ParamField header="x-api-key" type="string" required>
  Your API key for authentication
</ParamField>

## Path Parameters

<ParamField path="type" type="string" required>
  Type of trusted parties to retrieve. Must be one of: `domains`, `bank-accounts`, `mage-ids`
</ParamField>

## Response

The response contains an array with a key that matches the type:

<ResponseField name="domains" type="array" condition="when type=domains">
  Array of domain objects
</ResponseField>

<ResponseField name="bank_accounts" type="array" condition="when type=bank-accounts">
  Array of bank account objects  
</ResponseField>

<ResponseField name="mage_ids" type="array" condition="when type=mage-ids">
  Array of mage ID objects
</ResponseField>
